Introduction to HTML5

HTML5 is the fifth and current version of HTML. It has improved the markup available for documents and has introduced application programming interfaces (API) and Document Object Model (DOM).

Features:

  • New multimedia features including audio and video controls with <audio> and <video> tags.
  • New graphics elements like vector graphics and canvas tags.
  • Drag and Drop API allows users to drag objects across the web page.
  • Geolocation API enables websites to retrieve the geographical location of a client.
  • Web Storage provides methods to store data on the client-side, enhancing offline capabilities.
  • Support for SQL databases to store data offline.
  • Supports drawing various shapes such as triangles, rectangles, and circles using the canvas tag.
